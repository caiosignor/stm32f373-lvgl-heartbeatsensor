26 // ILI9341 GFX library for STM32
27 //-----------------------------------
28 //
29 // Very simple GFX library built upon ILI9342_STM32_Driver library.
30 // Adds basic shapes, image and font drawing capabilities to ILI9341
31 //
32 // Library is written for STM32 HAL library and supports STM32CUBEMX. To use the library with Cube software
33 // you need to tick the box that generates peripheral initialization code in their own respective .c and .h file
34 //
35 //
36 //-----------------------------------
37 // How to use this library
38 //-----------------------------------
39 //
40 // -If using MCUs other than STM32F7 you will have to change the #include "stm32f7xx_hal.h" in the ILI9341_GFX.h to your respective .h file
41 //
42 // If using "ILI9341_STM32_Driver" then all other prequisites to use the library have allready been met
43 // Simply include the library and it is ready to be used
44 //
45 //-----------------------------------
